# Project Title

Express backend API to 'Syndicated' React APP

## Description

Express API used manage data between React front end and firebase storage, database, and authentication

## Getting Started

run npm index.js in your terminal

### Dependencies

* node
* express
* Axios
* cors
* dotenv
* firebase
* multer
* uuid
